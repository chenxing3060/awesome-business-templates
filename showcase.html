多环境支持

同时支持浏览器和 Node.js 环境，UMD 模块格式

// Node.js 使用示例
const data = require('./templates.json');
const TemplateMatcher = require('./template-matcher.js');
const matcher = new TemplateMatcher(data);

// 智能推荐
const result = matcher.recommend({
  text: "一场关于春天花卉的摄影展，温暖浪漫的氛围",
  useCase: "gallery"
});
console.log(result.results); // Top 20
// 浏览器中使用（详见 index.html）

How to Use

快速上手

三步走：浏览 → 筛选 → 打开使用

1

浏览所有模板

打开 index.html，浏览全部 205 套模板。支持按流派、情绪、色彩、场景多维度筛选，也可使用关键词搜索。

2

智能匹配

在智能匹配面板中输入内容描述（如"一场春天的花展，温暖浪漫"），引擎自动推荐 Top 20 最合适模板。

3

打开使用

点击任意模板卡片即可在浏览器中打开对应的基座模板。键盘 ← → 翻页，直接替换内容即可。
